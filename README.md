# Hello Startup WordPress Plugin

A basic custom WordPress plugin created as a learning project.

## Features
- Displays a custom admin notice in WordPress dashboard
- Safe access check using ABSPATH
- Clean and minimal plugin structure

## How to Use
1. Copy this folder to `wp-content/plugins/`
2. Activate **Hello Startup Plugin** from WordPress admin
3. See the admin message appear

## Purpose
Learning WordPress plugin development and GitHub consistency.
